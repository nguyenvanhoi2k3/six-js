six-js — svgMotion demo

1. draw — line reveal (0% → 100%)

six.svgMotion.draw(target, vars) animates stroke-dasharray/dashoffset to draw a shape's outline in, out, or as a moving "comet" segment. Works on any single path, or a whole selector of shapes at once (staggered).

single path, 0% → 100%
3 paths, staggered ("SIX")
looping comet segment (from/to window)

2. morph — shape ↔ shape

six.svgMotion.morph(target, toShape, vars) animates a path's own d into another shape's - a <circle>, <path>, or any other basic shape all work as toShape, since non-path shapes are converted to an equivalent path automatically.

3. motionPath — move along an SVG path

six.svgMotion.motionPath(target, vars) moves a plain DOM element along a guide path, with optional autoRotate to face the direction of travel. Guide path shown dashed for reference.
