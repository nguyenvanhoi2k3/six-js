attrs of sx-dialog-trigger: - name (required): tên để định danh dialog, phải trùng với name của sx-dialog attrs of sx-dialog: - name (required): tên để định danh dialog, phải trùng với name của sx-dialog-trigger - duration (number): thời gian đóng mở dialog, mặc định là 300ms - close-on-outside-click (boolean): nếu true thì đong dialog khi click ra ngoài dialog và ngược lại, mặc định là true - close-on-esc-key (boolean): Cho phép đóng dialog bằng esc key, mặc định là true - scrollable (boolean): nếu true thì khi mở dialog background vẫn có thể scroll, mặc định là false (lưu ý nếu tắt scroll thì phải giữ nguyên width viewport không được để co giật width thanh scrollbar trình duyệt bị disable) - overlay (boolean): nếu true thì dialog có overlay, mặc định là true - overlay-style (string): style của overlay, mặc định là background-color: rgba(0, 0, 0, 0.5); (ví dụ: overlay-style="backdrop-filter: blur(10px);") tương lai phát triển sau, chưa làm vội: breakpoints
